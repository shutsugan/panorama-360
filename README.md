# panorama-360

A Panorama with the ability to create info pointers using ThreeJs

## Install

```shell
npm install -g http-server
```

clone the repo, cd in to it and run this command
```shell
http-server
```

to see the demo 
enter http://127.0.0.1:8080/ in the browser
